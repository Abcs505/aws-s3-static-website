# 🚀 AWS S3 Static Website Hosting

## 📌 Project Overview

A personal portfolio website built with HTML & CSS 
and hosted on Amazon S3 cloud storage.

## 🌐 Live Demo
[Click here to view live website](http://my-first-bucket-abhi-2025.s3-website.ap-south-1.amazonaws.com)

## ☁️ AWS Services Used
- **Amazon S3** — Static website hosting
- **IAM** — Secure access management
- **S3 Bucket Policy** — Public access configuration

## 🛠️ What I Learned
- Creating and configuring S3 buckets
- Enabling static website hosting on S3
- Managing bucket policies for public access
- Deploying a website to AWS Cloud
